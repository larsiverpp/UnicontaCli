# UnicontaCli

## Installation

- Download and install .NET Runtime 10, e.g. [.NET Runtime 10.0.2 for Windows X64](https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-10.0.2-windows-x64-installer).

## Usage

```
Liversen.UnicontaCli.exe --loginId MyLoginId --password MyPassword --accessIdentity MyAccessIdentity --companyId MyCompanyId inventory-stock-status get 2025-12-31
```
